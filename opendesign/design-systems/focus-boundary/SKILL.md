# FocusBoundary Design System Skill

This folder contains the complete visual and typography definitions for the physical paper planner aesthetic of FocusBoundary.

## Core Visual Theme
- **Cream Paper Base**: Warm off-white `#faf8f5` page with a radial dot-grid background resembling a physical bullet journal.
- **Graphite Outline Rules**: Crisp margins, thin borders, and graphite pencil pencil-stroke shadows simulating hand-drawn pages.
- **Ink Highlighter Accents**: Blue fountain pen ink (`#1d4ed8`) as the primary interactive accent, combined with teal, purple, green, and correction-red highlighting markers.

## File Hierarchy
- `./tokens/colors_and_type.css` — Standardized CSS Variables.
- `./README.md` — Visual brand guidelines and index of components.
