# FocusBoundary Design System

This design system defines the physical notebook aesthetic of FocusBoundary, a capacity planner and velocity boundary tool.

---

## 1. Visual Aesthetics & Metaphors
- **Bullet Journal Styling**: Emulates a real paper notebook using off-white cream page sheets, radial dot-grid backgrounds, and graphite lines.
- **Highlighter Highlights**: Status badges, point labels, and progress bars simulate dry ink markers (fountain blue, forest green, warm amber, and correction pen red).
- **Physical Depth**: Flat components styled with sharp borders and graphite drop shadows.

## 2. Typography pairing
- **Headings (Headers / Week Indicators)**: `Lora` (Serif) to mimic handwritten journal lettering.
- **Labels / Inputs / Copy**: `Inter` (Sans-serif) for legibility and clear user interface interaction.
- **Stats / Codes / Meta-Badges**: `JetBrains Mono` (Monospace) to convey structural precision.

## 3. Brand Tone
- Supportive, objective, and clear. Avoid neon alerts, flashing popups, or anxiety-inducing notification counters. Protect user focus first.
